📡 Ping Coming Soon Page

A responsive Coming Soon Landing Page built as part of a Frontend Mentor challenge.
The project focuses on building a clean UI with an email subscription form and validation.

🚀 Features

- Clean single-column layout
- Email subscription input field
- Form validation (empty + invalid email)
- Error message handling
- Social media icons section
- Responsive layout (mobile → desktop)

| Technology             | Purpose            |
| ---------------------- | ------------------ |
| **HTML5**              | Page structure     |
| **CSS3**               | Styling and layout |
| **JavaScript**         | Form validation    |
| **Flexbox / CSS Grid** | Layout alignment   |
| **GitHub Pages**       | Deployment         |
